# Build 031 - Belt Rank API Module

## Added

- Create Belt Rank API.
- Get all Belt Ranks API.
- Get Belt Rank by id API.
- Update Belt Rank API.
- Activate Belt Rank API.
- Deactivate Belt Rank API.

## Endpoints

- POST `/api/belt-ranks`
- GET `/api/belt-ranks`
- GET `/api/belt-ranks/{id}`
- PUT `/api/belt-ranks/{id}`
- PATCH `/api/belt-ranks/{id}/activate`
- PATCH `/api/belt-ranks/{id}/deactivate`

## Notes

Belt Rank is required before building promotion, belt history, attendance reporting and student filtering by belt.
